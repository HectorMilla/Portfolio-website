Portfolio

  • SOC Analyst Home Lab

    In this project, I utilized VMware to set up a virtualization environment with Linux and Windows VMs. I created a C2 payload to simulate an attack on the Windows VM and implemented Sysmon and LimaCharlie as an EDR agent for monitoring and response. I simulated an lsass.exe dump to capture the attack on LimaCharlie and created a detection rule for this event. Additionally, I performed an attack to delete volume shadow copies and created a blocking rule in LimaCharlie to prevent this action. The project aimed to assess the effectiveness of security measures, improve threat detection, and enhance response capabilities in the virtualized environment.

  • Windows Server 2016 | Hands-On Virtual Lab

    In this project, I set up a virtualized network environment using VirtualBox, pfSense, and Windows Server 2016. By joining Windows 10 clients to the domain, I centralized user and resource management. Active Directory template user accounts were customized, and Group Policy Objects (GPO) were created to enforce network-wide settings. I configured server settings, TCP/IP, and remote desktop access. Additionally, a Bash script was developed to automate the creation of 1,000 simulated users. Overall, this project established an efficient network infrastructure for managing users, resources, and security.
